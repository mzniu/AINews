# AI资讯视频生成器 - 项目概述

## 项目简介
自动化AI资讯聚合系统，通过爬取多个AI资讯网站的最新内容，利用DeepSeek API进行智能总结，并自动生成短视频内容。

## 核心功能
1. **资讯爬取** - 多源AI新闻网站爬虫
2. **内容总结** - DeepSeek API智能摘要
3. **视频生成** - 自动化短视频制作

## 目标用户
- AI从业者和爱好者
- 需要快速了解AI行业动态的用户
- 短视频平台内容创作者

## 技术价值
- 自动化内容聚合，节省人工筛选时间
- AI驱动的内容总结，保证质量和准确性
- 批量视频生成，提高内容产出效率

## 项目目标
- 每日自动抓取和更新AI资讯
- 生成3-5分钟的资讯总结视频
- 支持多平台发布（抖音、B站、YouTube等）
